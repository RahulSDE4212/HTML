This is the showcase of list

list tag in html are of two types:

  1. unordered list
  2. ordered list

unordered list

unordered list is used to list the element in the docs in unordered way
for example
  • list 1
  • list 2
  • list 3
  • list 4

ordered list

ordered list is used to list the element in the docs in numbered way
for example
  1. list 1
  2. list 2
